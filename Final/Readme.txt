使用说明：

1.双击SheepTools.exe运行分类工具
2.参照输入文件格式，加载训练集和测试集数据
3.选择分类算法并配置相关参数
4.选择参与分类计算的属性，默认所有属性均参与分类
5.点击“开始”执行分类算法
6.查看分类结果与结果评估
7.执行产生的分类和评估结果将自动保存至本地./user/Result.txt和 ./user/Evaluation.txt文件。


注意：

1.在win7/win8系统下，若将程序安装至系统盘，请务必以管理员身份运行！
2.Data Set Samples文件夹中提供了六组样例数据，供测试使用
3.所有参与分类的样本属性类型均为整数或实数，若为其他类型，请转化成对应整数值
4.若输入测试集样本类别未知，请为每一样本随意添加一个类别值使程序能够正常工作，添加值不影响分类结果。
5.性能评估中，错误率只针对类别已知的样本有效。
